1. Descripción general

El sistema Publicidad RELCO reproduce automáticamente videos e imágenes en pantalla completa sobre el monitor secundario al iniciar Windows. Está formado por un único script de PowerShell (vlc-publicidad.ps1) que gestiona la instalación, los accesos directos y el lanzamiento de VLC Media Player.

2. Requisitos previos

  • Windows 10 / 11
  • VLC Media Player instalado (64 o 32 bits)
  • Monitor secundario conectado y reconocido por Windows en modo Extender (puede aparecer como Ampliar según la versión de Windows)
  • PowerShell 5.0 o superior (incluido en Windows 10/11)

3. Estructura de carpetas

El script espera que los archivos de publicidad estén en:

C:\Users\Public\relco_videos\

Si la carpeta no existe, el script la crea automáticamente en el primer arranque e indica al operador que debe agregar contenido.

Formatos soportados

ExtensiónTipoNotas
.mp4VideoH.264 recomendado
.jpg / .jpegImagenSe muestra 10 seg. por defecto
.pngImagenSe muestra 10 seg. por defecto

Los archivos se ordenan alfabéticamente por nombre. Para controlar el orden de reproducción, anteponga un prefijo numérico: 01_bienvenida.mp4, 02_oferta.jpg, etc.

4. Habilitar ejecución de scripts en PowerShell

Windows bloquea la ejecución de scripts de PowerShell por defecto. Antes de instalar, ejecute una sola vez el siguiente comando en PowerShell:

Opción A — Usuario actual (no requiere Administrador):
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

Opción B — Todos los usuarios (requiere abrir PowerShell como Administrador):
Set-ExecutionPolicy RemoteSigned

Confirme con S cuando PowerShell lo solicite.

Verificación: puede comprobar que quedó aplicado ejecutando Get-ExecutionPolicy -List

5. Instalación

Pasos (única vez):
  1. Instale VLC Media Player antes de continuar. El script lo busca en el sistema al ejecutarse; si no lo encuentra, la instalación falla.
    Descarga: videolan.org/vlc
  2. Copie vlc-publicidad.ps1 a cualquier carpeta accesible.
  3. Abra PowerShell y navegue hasta esa carpeta:
    cd C:\ruta\donde\guardó\el\script
  4. Ejecute el script con el parámetro -Setup:
    .\vlc-publicidad.ps1 -Setup
Importante: El script solo se instala cuando se ejecuta con -Setup desde PowerShell. Sin ese parámetro únicamente lanza VLC (es el modo que usan los accesos directos ya instalados).

Durante la instalación el script:

  • Verifica que VLC esté instalado (abre la página de descarga si no lo encuentra).
  • Se copia a C:\Users\Public\vlc-publicidad.ps1.
  • Crea un acceso directo "Publicidad RELCO" en el Escritorio.
  • Crea un acceso directo en Inicio automático del usuario actual, para que arranque con Windows.
  • Abre el panel de selección de pantalla (igual que presionar Win + P): seleccione Extender (o Ampliar) para activar el monitor secundario.
Nota: Una vez completada la instalación, use el acceso directo del escritorio para iniciar la publicidad.

6. Acceso directo en el Escritorio

Tras la instalación aparece el icono "Publicidad RELCO" en el escritorio. Úselo para:

  • Relanzar la publicidad después de cerrarla manualmente.
  • Aplicar cambios de playlist: cada vez que agregue, quite o renombre archivos en relco_videos\, haga doble clic en este acceso directo para que VLC cargue la nueva lista.
Tip: VLC no detecta cambios en la carpeta en tiempo real. Siempre use el acceso directo del escritorio para reiniciar la reproducción con el contenido actualizado.

7. Icono en el área de notificaciones (reloj)

Cuando la publicidad está activa, VLC aparece como un pequeño icono de cono naranja en el área de notificaciones (junto al reloj, esquina inferior derecha de la pantalla).

  • Clic derecho → Salir — cierra VLC completamente.
  • Si el icono no es visible, haga clic en la flecha "Mostrar iconos ocultos" (el símbolo ^ junto al reloj).
Nota: La ventana de VLC está oculta de la barra de tareas de forma intencional para no interferir con el escritorio. El único punto de acceso para cerrarlo es el icono del área de notificaciones o el Administrador de tareas.

8. Desactivar notificaciones de Windows para VLC

Windows puede mostrar notificaciones grises (globos o banners) generadas por VLC que interrumpen la pantalla de publicidad. Para desactivarlas:

  1. Abra Configuración de Windows (tecla Windows + I).
  2. Vaya a Sistema → Notificaciones y acciones.
  3. En la lista de aplicaciones busque VLC media player y desactive el interruptor.
Tip: Si prefiere desactivar todas las notificaciones de una vez, apague el interruptor principal "Recibir notificaciones de aplicaciones y otros remitentes" en la misma pantalla.

9. Actualizar el contenido de publicidad

  1. Copie los nuevos archivos (.mp4, .jpg, .png) a C:\Users\Public\relco_videos\.
  2. Elimine o renombre los archivos que ya no deban reproducirse.
  3. Cierre VLC desde el icono del área de notificaciones (clic derecho → Salir).
  4. Haga doble clic en el acceso directo "Publicidad RELCO" del escritorio para reiniciar con la nueva lista.

10. Parámetros configurables

Los siguientes valores se pueden ajustar editando las primeras líneas del script:

VariableValor por defectoDescripción
$MediaFolderPublic\relco_videosCarpeta de contenidos
$ImageDuration10 (segundos)Tiempo por imagen
$SecondaryScreen10 = primaria · 1 = secundaria

Tras cualquier cambio en el script, vuelva a ejecutarlo con -Setup para reinstalar los accesos directos.

11. Solución de problemas

SíntomaCausa probableSolución
VLC no arranca al encender el PC Acceso directo de inicio automático no creado Ejecutar el script con -Setup
Pantalla negra / sin contenido Carpeta relco_videos vacía Agregar archivos y reiniciar con el acceso directo
El video sale en la pantalla incorrecta Valor de $SecondaryScreen Cambiar a 0 o 1 según corresponda
La publicidad aparece en la pantalla principal o en ambas Windows no quedó en modo Extender Presione Win + P y seleccione Extender (o Ampliar). La instalación lo configura automáticamente, pero algunos equipos revierten el modo al reiniciar.
Las imágenes se ven muy poco tiempo $ImageDuration bajo Aumentar el valor en el script y relanzar
VLC aparece en la barra de tareas Error al ocultar la ventana Cerrar y relanzar desde el acceso directo
VLC no se encuentra VLC no instalado o en ruta no estándar Reinstalar VLC Media Player

12. Desinstalación

  1. Cierre VLC desde el icono del área de notificaciones.
  2. Elimine el acceso directo del Escritorio: Publicidad RELCO.lnk
  3. Elimine el acceso directo de inicio automático en:
    C:\Users\<usuario>\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup\
  4. Elimine C:\Users\Public\vlc-publicidad.ps1 si lo desea.
